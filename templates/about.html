Website Description

This Website is made for the prupose of DBMS project. The main aim was to develop an interactive webapp or a mobile app for Human Resource Management. The Website maintains database of all the emmployees in a company. It can maintain attendance and incentive info forr all the days for an employee. IT can also generate Salary for employee between any two specified dates. Any new employee can bew added by the admin and only admin can mark attendance, incentive of any employee. Salary information also can be generated by an admin only. All the employees can view their attendance and incentive through dashboard and also view their profile and edit it through profile tab.
